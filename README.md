# ⚡ MINER PRO

> Mine. Invest. Earn.

MINER PRO is a Telegram-based play-to-earn mining game built on the TON blockchain ecosystem.

## 🤖 Bot
[@Minerbyner_bot](https://t.me/Minerbyner_bot)

## 🎮 How it works
- Mine TRX by tapping and boosting hashrate
- Buy 12 upgradeable mining items
- Build a binary referral network
- Invest in TON miners for daily passive income
